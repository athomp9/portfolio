Tying Up Loose Ends: DOMINATION; submission

This print was my first in letterpress.

Below you can see the process of this project. I conversed with ink mixing, embossing strategies, and perpetual proofing.

I needed to make sure everything was in the right spot. "submission" fitting perfectly in the counter of the 'O' I used as an ornament was essential to this project and its concepts.

This project was an intense learning curve for me. I couldn't just move things around a screen, and I had to manually adjust every aspect of what ends up on the page. This is something I fell in love with.
